Introdução¶

O desenvolvimentos de mtodos de mofologia matemtica baseiam-se em teoridas de conjuntos da matemtica. Esta abordagem considera a imagem como o conjunto universo, objetos com conjuntos contidos no conjunto universo e estuda o resultado de operadores de deslocamento livre sobre a imagem. As operaes morfolgica visam promover melhoramentos na imagem, como remoo de rudo, juno de elementos separados, achar vales e picos em uma imagem. Alguns dos mtodos so os de (i) eroso e (ii) dilatao, so ditos como elementares neste mbito.

A operao de eroso consiste na alterao morfolgica da imagem e sua operao computa o mnimo local dentro da rea do elemento estruturante enquanto que a dilatao

Discussões sobre os métodos¶

O mtodos morfolgicos so simples operaes baseadas na forma do imagem, cujo objetiva obter melhorasmentos da mesma. A entrada para o mtodo a imagem propriamente dita (no exemplo a seguir utilza-se uma imagem binria para melhor entendimento) e o elemento estruturante, que ir operar sobre a imagem, bit a bit.

Os mtodos implementados a seguir so os mtodos da eroso e dilatao, utilizando trs elementos estruturantes distintos.

A eroso um mtodo que promove diminuio os limites do objeto e do seu fundo.

def EE (size, type):
    if type == 1:
        # square
        kernel = np.ones(size, np.uint8)
        return kernel
    if type == 2:
        # star
        kernel = np.ones(size, np.uint8)
        center = np.uint8((size[0]/2))
        for k in range(0,size[0]):
            kernel[center,k] = 0
            kernel[k,center] = 0
        return kernel
    if type == 3:
        # something 01
        kernel = np.ones(size, np.uint8)
        center = np.uint8((size[0]/2))
        kernel[:,0] = 0
        kernel[center,1] = 0
        return kernel

Dualidade entre erosão e dilatação¶

Utilizando a mesma imagem, porm com as cores invertidas fcil visualizar a aplicao dos mtodos:

  • A eroso promove uma diminuio da regio branca, logo um aumento do objeto
  • A dilatao promove um aumento da regio branca, uma diminuio do objeto.

Conclusão¶

Os mtodos morfolgicos de eroso e dilatao so operadores elementares que funcionam de forma dual, dependendo da imagem em questo. Em imagens binrias promovem diminuio e crescimento de regies brancas respectivamente, e espera-se que em imagens tons de cinza promovam efeitos semelhante a filtros passa-alta a e passa-baixa, removendo rudos, ressaltando bordas e outras operaes.
